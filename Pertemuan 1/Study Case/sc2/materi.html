Materi Pembelajaran

1. Konsep dasar HTML

  a. Tag HTML dasar

HTML memilki beberapa tag yang umum digunakan diantaranya <body>, <p>, <h1>, <img>, <table>, dll.

  b. Struktur dasar dokumen HTML

Dalam struktur dasar HTML, kita memulai dengan mendeklarasikan tipe dokumen HTML yang digunakan (pada contoh di atas, menggunakan tipe dokumen HTML versi terbaru yaitu HTML5). Kemudian, kita menetapkan elemen induk dokumen HTML yaitu <html> yang akan berisi elemen-elemen lainnya. Bagian <html> kemudian dipecah lagi menjadi dua bagian yaitu <head> dan <body>.
