Deskripsi bahan ajar

Bahan ajar ini bertujuan untuk memperkenalkan konsep dasar pengembangan situs web secara sederhana kepada pemula. Materi dalam bahan ajar ini akan meliputi konsep dasar HTML, CSS, dan JavaScript serta cara membuat website sederhana dengan menggunakan bahasa pemrograman tersebut.
